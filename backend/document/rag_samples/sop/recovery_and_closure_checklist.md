# SOP: Recovery and Closure Checklist

- Version: 1.0

- Effective Date: 2026-04-17
- Language: en

## 2. Purpose
Define a compact checklist for moving from a protected posture back to normal intake after a salinity event.

## 3. Closure Preconditions
- Three consecutive readings must remain below the warning band.
- No fast-rise signal should appear in the latest review window.
- Tide and wind modifiers should no longer support an intrusion posture.
- Sensor confidence should be acceptable for the station that triggered the incident.

## 4. Staged Reopening
- Step 1: partial reopening or limited operation.
- Step 2: confirm stability during the next reassessment cycle.
- Step 3: resume full normal operation only after stability is sustained.

## 5. Communication and Records
- Notify operators and affected users before each posture change.
- State whether the action is simulation-only or a real operational change.
- Record the decision rationale, closure evidence, and follow-up timestamp.
- Create a memory case after closure so future similar events can be retrieved.

## 6. Failure Modes
- If new data arrives with lower confidence, extend the observation window.
- If the tide turns adverse again, revert to protection posture.
